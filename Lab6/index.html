Preguntas

¿Por qué es una buena práctica usar JavaScript para checar que sean válidos los inputs de las formas antes de enviar los datos al servidor?

Porque de esta manera nos ahorramos inconveniencias desde el inicio y podemos alertar al usuario si es que algún dato es erróneo, dando así una mejor experiencia de usuario.

¿Cómo puedes saltarte la seguridad de validaciones hechas con JavaScript?

Hay varias maneras pero puedes simplemente deshabilitar javascript en el navegador.

Si te puedes saltar la seguridad de las validaciones de JavaScript, entonces ¿por qué la primera pregunta dice que es una buena práctica?

Porque en la mayoría de los casos si se ocupa tener la validación de JavaScript para el usuario. Pero hay veces que no necesariamente sea un requisito y pueda saltarse.
