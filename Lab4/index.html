Preguntas

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Ambos son lenguajes de programación totalmente diferentes pero hay varias diferencias entre ambos.

JavaScript es un lenguaje interpretado, mientras que Java compilado. Esto quiere decir que JS son archivos de texto integrados en las páginas HTML e interpretados por un navegador. En cambio, en java se compila a un archivo especial para poder ser ejecutado por un ordenador. Estos dos lenguajes no tienen nada que ver juntos, hay varias diferencias entre ellas por lo que es importante diferenciar ambas.

Por otro lado, estos dos lenguajes cuentan con ciertas similitudes dado a que tienen una sintaxis similar a C. Además de que cuentan con estructuras de control y delimitadores muy parecidos.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

  1. getMinutes() Devuelve los minutos (de 0 a 59)
  2. now() Devuelve el número de milisegundos desde medianoche del primero de enero de 1970
  3. setFullYear() Establece el año de un objeto de fecha
  4. setTime() Establece la fecha a un número específico de milisegundos antes/después del primero de enero de 1970
  5. valueOf() Devuelve el valor de un objeto Date

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

  1. concat() Une dos o más matrices y devuelve una copia de las matrices unidas
  2. forEach() Llama a una función para cada elemento de la matriz.
  3. map() Crea una nueva matriz con el resultado de llamar a una función para cada elemento de la matriz
  4. pop() Elimina el último elemento de una matriz y devuelve ese elemento
  5. copyWithin()Copia elementos de la matriz dentro de la matriz, hacia y desde posiciones especificadas

¿Cómo se declara una variable con alcance local dentro de una función?

Utilizando let en vez de var, ya que let se acaba en cuanto se acaba la ejecucion de bloque. Esto funciona para while,for,etc.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
